# 🎓 Student Productivity Data Analysis

A data analysis project exploring factors that influence student productivity and exam performance using Python and Pandas.

---

## 📌 Project Overview

This project analyzes a student productivity dataset containing 5000 records.  
The goal is to understand how different lifestyle and academic factors affect exam performance.

Key focus areas include:
- Study hours
- Gaming and social media usage
- Sleep hours
- Internet quality
- Academic level

The project applies data cleaning, exploratory data analysis (EDA), grouping, masking, correlation analysis, and visualization to uncover insights.

✔ Data inspection and cleaning

✔ Correlation analysis

✔ Relationship visualization using Matplotlib  

---

## 🔍 Key Questions Explored

- Does study time improve exam performance?
- Does gaming or social media reduce exam scores?
- Does internet quality influence productivity?
- Which academic level performs best?
- What factors are most correlated with exam scores?

---

## 📈 Sample Insights 

- Internet quality showed minimal impact on exam scores.
- Gaming and social media usage showed negative correlation with performance.
- Study hours had a positive relationship with exam score.
- Differences between academic levels were small but measurable.
